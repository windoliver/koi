## Key Interactions to Verify

- Session replay: select session → view events → paginate → filter by type

- Health check: run → verify each source → display green/red per item

## Edge Cases
- GDrive OAuth: user closes browser mid-auth (should timeout + retry prompt)
- GDrive OAuth: corporate proxy blocks localhost callback (manual token fallback)
- GDrive: large shared drive with 10,000+ files (lazy loading, not full scan)
- Session replay: active session (live-updating badge, not stale data)
- Session replay: session with 50,000+ events (cursor pagination)
- Compliance skill: email with prompt injection attempt (must flag, not clear)
- Compliance skill: non-English email (flag as "unsupported language")
- Demo seed: run twice (must be idempotent)
- Health check: one source offline, others fine (show mixed status, not global fail)
- GDrive: OAuth token revoked after 6 months inactivity (trigger re-auth wizard)
